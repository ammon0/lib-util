size_t size_t strlcat ( char *restrict  dst,
const char *restrict  src,
size_t  dstsize 
)

Appends the string in src to the end of dest It will append at most dstsize - strlen(dst) - 1 bytes.

It will then NUL-terminate, unless dstsize is 0 or the original dst string was longer than dstsize.

  • guarantees that dst will be '\0' terminated
  • If the src and dst strings overlap, the behavior is undefined.
    Parameters
    dstthe region of memory the string is to be copied to
    srcthe null terminated string to be copied
    dstsizethe size in bytes of dst
    Returns
    the initial length of dst plus the length of src. If return value >= dstsize then the src string has been truncated.
