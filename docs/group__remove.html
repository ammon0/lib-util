Detailed Description

Remove a data entry from a known position.

The returned pointer points to the data stored in a temporary space. The caller must copy that data out as there is no guarantee that it will be preserved after the next call to the data structure.

If your data structure is a queue use only DS_nq() and DS_dq(). If your data structure is a stack use only DS_push() and DS_pop().

a pointer to the removed data on success, NULL on failure.
