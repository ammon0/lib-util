Detailed Description

Functions for logging and printing messages to stderr.

Log Levels

The log level setting determines what messages will be printed to stderr. All messages are printed to the given log stream with a tag indicating the level of the message.

Log Descriptors

The caller may create as many log descriptors as needed, each corrosponding to a different log file.
