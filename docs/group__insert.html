Detailed Description

Insert new data into an existing structure.

Data is copied into the structure. The size of the data must be the same as the data_size parameter used when the structure was initialized.

If your data structure is a queue use only DS_nq() and DS_dq(). If your data structure is a stack use only DS_push() and DS_pop()

Parameters
rootis the root of a data structure
datais a pointer to the data being inserted
Returns
NULL on failure. Otherwise it returns a pointer to the inserted data in its new location. They may fail if they are unable to allocate more memory.

Insert data in sort order, or at the current position

For sorted structures DS_insert() will insert the data in sort order. For other structures the data will be inserted at the current position. This call will fail on a sorted structure if a duplicate key is found and duplicates are not allowed.

rootis the root of a data structure
